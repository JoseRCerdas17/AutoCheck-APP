# AutoCheck API

API desarrollada con NestJS para la gestión de usuarios y vehículos.

## Tecnologías utilizadas

- Node.js
- NestJS
- TypeScript
- SQL Server
- TypeORM

## Requisitos

Antes de ejecutar el proyecto debes tener instalado:

- Node.js
- npm
- SQL Server
- Git

## Clonar el repositorio

git clone https://github.com/JoseRCerdas17/AutoCheck-APP.git

cd AutoCheck-APP

## Instalar dependencias

cd backend

npm install

## Ejecutar el proyecto

npm run start:dev

El servidor se ejecutará en:

http://localhost:3000git
